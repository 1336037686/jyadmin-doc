# 简介



**jyadmin** 是一个基于Spring Boot 2.X、Mybatis Plus、Spring Security、Vue、Element-UI 的前后端分离的后台管理系统。



>  🌳 初衷

为开发者提供一套快速开发的脚手架。通过完善的基础功能，为后台管理系统的定制开发提升效率。
